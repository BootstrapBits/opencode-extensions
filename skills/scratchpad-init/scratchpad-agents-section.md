# Scratchpad Section for AGENTS.md

This file contains the section to add to a project's AGENTS.md file when initializing a scratchpad.

---

## Scratchpad Usage

This project uses a `scratchpad/` directory for persistent context sharing between agent sessions.

### Before Planning or Task Work

1. Read `scratchpad/scratchpad-summary.md` first
2. Review which files might be relevant to your task
3. Ask the user before reading full scratchpad documents
4. Use the context to inform your approach

### After Completing Work

1. Update `scratchpad/STATUS.md` with current progress
2. Update `scratchpad/scratchpad-summary.md` to reflect changes
3. Create dated notes for significant decisions: `scratchpad/YYYY-MM-DD-topic.md`

### File Naming Convention

- Core docs: `UPPERCASE.md` (PLAN, CONTEXT, IDEAS, STATUS)
- Dated notes: `YYYY-MM-DD-topic-name.md`
